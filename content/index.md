## 🔄 Deployment Workflow

Setiap ada update catatan di Obsidian, jangan lupa untuk melakukan sinkronisasi agar website ini terupdate otomatis:

1.  Tulis/Edit file di **Obsidian**.
2.  Buka terminal di folder `~/quartz`.
3.  Jalankan perintah sakti:
    ```bash
    npx quartz sync
    ```
