cp -i /etc/NetworkManager/NetworkManager.conf .

  

parameter diatas mencopy sebuah file networkmanager.conf di dalam folder /etc/ ke current directory dimana terminal ada. karena fungsi ( . ) di akhir sebuah perintah menandakan untuk di copy kesini. serta parameter -i adalah interactive. dimana ketika file sudah ada di current directory, sistem akan memberikan peringatan untuk me-replace atau tidak file tersebut.
